Trabalho 3 - Aplicação de Calculadora em Flutter utilizando Componentização 

1 - Descrição da Aplicação:

Este projeto consiste em uma calculadora simples utilizando o framework Flutter, com interface gráfica interativa.
A aplicação permite realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão, além de exibir os resultados em tempo real na tela, e fazer a limpeza da operação.

2 - Explicação da Estrutura de Componentes:

O projeto foi dividido em widgets reutilizáveis para melhor organização e manutenção do código.

- Widget Botão Numérico: Responsável por representar os números da calculadora.
- Widget Botão de Operação: Representa as operações matemáticas - adição (+), subtração (-), multiplicação (×), divisão (÷).
- Widget Display: Responsável por exibir - números digitados, operações e resultado final.
- Organização do Layout : A interface foi construída utilizando - Row, Column, Expanded, Container.
